# 🏥 Clinic Management System

Capstone Project II

A web-based system designed to support clinic operations including patient management, appointment scheduling, treatment records, and service management.

---

## 🚀 Tech Stack

| Layer    | Technology        |
| -------- | ----------------- |
| Frontend | Next.js           |
| Backend  | Node.js + Express |
| Database | PostgreSQL        |

## ⚙️ Project Setup

### 1. Clone Repository

```bash
git clone https://github.com/your-repo/clinic-management-system.git
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Development Server

```bash
npm run dev
```

The application will run at:

```
http://localhost:3000
```
